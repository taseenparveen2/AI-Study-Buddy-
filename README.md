
📚 AI Study Buddy

AI Study Buddy is a simple AI chatbot that helps users:

Understand concepts
Generate quizzes
Solve doubts
Get summaries

It uses multiple AI agents and real-time web search.

🚀 Features
🤖 Explanation Agent – explains topics clearly
❓ Quiz Agent – generates MCQs
💬 Doubt Agent – answers follow-up questions
📝 Summarizer Agent – gives short summaries
🔍 Tavily Search – fetches real-time data
⚡ Groq LLM – fast AI responses
🎛️ Streamlit UI – simple interface
🧱 Tech Stack
Python
LangChain
Groq (LLaMA model)
Tavily API
Streamlit
